Feature details

Register Device

  • Scans for nearby Wacom SmartPad devices over Bluetooth LE
  • Supports Bamboo Spark, Bamboo Slate, and Bamboo Folio
  • Saves device address so subsequent launches connect automatically
  • No Wacom drivers or proprietary software required

Sync Offline Drawings (Normal Mode)

  • Pulls all drawings stored on the device since the last sync
  • Drawings are stored locally as JSON and rendered in tabbed view
  • Each tab shows the drawing date and time
  • Tabs can be closed individually without deleting the file on disk
  • Reload drawings from disk at any time with the Fetch button

Live Mode

  • Streams pen strokes to screen as you write — no sync step needed
  • Full-screen canvas with no tabs; orientation respects your current setting
  • Useful for digital annotation, presentations, or real-time note capture

SVG Export

  • Export any synced drawing as a scalable vector graphic (SVG)
  • File save dialog lets you choose location and filename
  • Compatible with Inkscape, Adobe Illustrator, browsers, and more

Portrait & Landscape Orientation

  • Default is Portrait; switch to Landscape without restarting
  • Portrait mode applies a 90° clockwise coordinate rotation so vertical pages display correctly
  • Orientation setting applies to new drawings and the Live canvas; existing tabs are unaffected

Per-Drawing Toolbar

  • Save SVG button in every tab for one-click export
  • Delete button permanently removes the drawing file from disk and closes the tab

Zero Install, Zero Cloud

  • Ships as a single portable EXE — copy it anywhere and run it
  • No installation, no admin rights, no registry entries
  • All data stays on your machine — no cloud account required
  • Open source under GPLv2
