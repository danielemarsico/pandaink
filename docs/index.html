PandaInk

Your Wacom SmartPad drawings, alive on Windows

A native Windows port of the open-source Tuhi project. Sync offline drawings from your Bamboo Spark, Slate, or Folio over Bluetooth LE and export them as crisp SVG files.

Everything you need, nothing you don't

PandaInk gives you a clean, focused desktop app that speaks directly to your smart pad.

Register Device

Pair your Wacom Bamboo Spark, Slate, or Folio via Bluetooth LE in a single click. No drivers, no proprietary cloud required.

Sync Offline Drawings

Drawings made while the app was closed are stored on the device. Hit Listen to pull them all down at once.

Live Mode

Stream pen strokes in real time directly to your screen as you write or sketch. Great for demos and digital inking workflows.

SVG Export

Every drawing can be saved as a clean, scalable SVG file — ready for Inkscape, Illustrator, or any vector workflow.

Portrait & Landscape

Switch orientation per session. Portrait mode rotates coordinates 90° so vertical pages look exactly right on screen.

Zero Install

The portable EXE runs without installation. No admin rights needed, no leftover registry entries.
